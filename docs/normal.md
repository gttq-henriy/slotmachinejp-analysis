# 通常時解析

## 通常時の抽選

通常時はスピンごとに小役を抽選し、その結果に応じてCZ抽選やAT直撃抽選が行われます。

### 小役確率（通常時）

総抽選重み: **65,536**

| 小役 | 重み | 確率 |
|------|------|------|
| Charcoal（炭） | 3,000 | 4.58% |
| Wood（木材） | 2,200 | 3.36% |
| Rope（ロープ） | 2,400 | 3.66% |
| LowGrade×1 | 700 | 1.07% |
| LowGrade×3 | 400 | 0.61% |
| Bandage（包帯） | 2,400 | 3.66% |
| Sulfur（硫黄） | 480 | 0.73% |
| Scrap | 220 | 0.34% |
| Lucky Seven | 6 | 0.009% |
| **ハズレ** | **53,730** | **81.98%** |

::: tip
設定差は小役確率にはありません。全設定共通です。
:::

## レア役からの抽選

### チェリー（Sulfur）/ スイカ（Scrap）成立時

→ 内部モードに応じた確率で **CZ抽選** が行われます（[CZ解析](/cz) 参照）。

### レア役成立時

通常時にレア役が成立した場合:
- **AT直撃**: 4%
- **CZ突入**: 60%
- **なし**: 36%

## 前兆 (Forewarning)

CZやATに当選すると **前兆演出** が2〜8ゲーム発生します。

| 項目 | 値 |
|------|-----|
| 最小ゲーム数 | 2G |
| 最大ゲーム数 | 8G |
| フェイク前兆発生率 | 10% |

::: tip
前兆中はLCD演出が変化します。前兆が発生しても、10%の確率でフェイク（ハズレ）の場合があるため注意。
:::
