# 天井・やめどき

## 天井

| 項目 | 値 |
|------|-----|
| 天井ゲーム数 | **800G**（全設定共通） |
| 恩恵 | 天井CZ突入 |
| リセット条件 | AT終了後 / 設定変更時 |

通常時800Gハマりで天井到達し、**天井CZ** に突入します。通常CZと同じルール（T3/T4によるゲーム数・基本率決定）で進行します。

::: warning
天井CZは通常CZと同様に失敗する可能性があります。ただしCZ中のチェリー/スイカ/レア役で成功率が上乗せされるため、実質的な成功率は高めです。
天井は通常時消化ゲーム数でカウントされます。CZ中やAT中のゲーム数はカウントされません。
:::

## 天井到達の期待値

天井到達時はCZに突入します。CZが成功すればATに入れます。

### AT1回あたりの期待獲得

AT初期40G中に7揃い（39.67%）で15倍配当（45 scrap）が期待できます。

- 40Gあたりの7揃い期待回数: 40 × 0.3967 ≒ **15.9回**
- 7揃い配当: 15倍 × 3 scrap = **45 scrap/回**
- AT1セットの期待獲得: 15.9 × 45 ≒ **715 scrap**（ベット×1時）
- AT1セットのコスト: 40 × 3 = **120 scrap**
- AT1セットの純増: 約 **595 scrap**

::: tip
セット継続やAddOn上乗せが発生すれば、さらに獲得量は増加します。
:::

## やめどきの目安

### AT終了直後

AT終了後は **T10テーブル** により、高設定ほど上位モードに滞在しやすくなります。

| 設定 | High以上モード率 |
|------|----------------|
| 設定1 | 15% |
| 設定2 | 24% |
| 設定3 | 34% |
| 設定4 | 45% |
| 設定5 | 58% |
| 設定6 | 68% |

**推奨**: AT後はモード優遇があるため、少なくとも **CZ1回挑戦か100G** まで継続推奨。

### 通常時のやめどき判断

| ハマりG数 | 推奨行動 |
|----------|---------|
| 0〜200G | AT後なら続行。モード恩恵が残っている可能性あり |
| 200〜500G | 低設定疑いなら撤退候補。他の判別要素と総合判断 |
| 500〜700G | 天井が近いため続行推奨 |
| 700G〜 | **絶対に続行**。あと100G以内に天井到達 |

::: warning 重要
**500G以上ハマっている場合は絶対にやめないでください！** 天井800Gまであと少しです。天井CZは失敗の可能性もありますが、投資分回収のチャンスです。
:::

## 投資額の目安

ベット倍率×1（コスト3 scrap/G）の場合:

| 到達G数 | 累計投資 |
|---------|---------|
| 100G | 300 scrap |
| 200G | 600 scrap |
| 400G | 1,200 scrap |
| 600G | 1,800 scrap |
| 800G（天井） | 2,400 scrap |

::: info
途中のCZ突入やLowGrade Face配当によって、実際の投資額はこれより少なくなります。
:::

## フリーズの期待値

| 項目 | 値 |
|------|-----|
| 発生確率 | 1/8,192 ≈ 0.012%（スピンあたり） |
| 恩恵 | 200G AT確定 |
| 200G ATの期待獲得 | 約 **3,575 scrap**（7揃いのみ計算、ベット×1時） |

## ベット倍率の影響

ベット倍率を上げるとコストも配当も同じ倍率で増加します。RTP（出玉率）自体は変わりませんが、**ハイリスク・ハイリターン** となります。

| ベット倍率 | スピンコスト | 天井投資（800G） | AT1セット期待純増 |
|-----------|------------|----------------|-----------------|
| ×1 | 3 scrap | 2,400 scrap | 約595 scrap |
| ×3 | 9 scrap | 7,200 scrap | 約1,785 scrap |
| ×5 | 15 scrap | 12,000 scrap | 約2,975 scrap |
| ×10 | 30 scrap | 24,000 scrap | 約5,950 scrap |

::: tip
低設定狙いの場合は低ベットでコツコツ、高設定確信時はベットを上げて一気に回収するのが基本戦略です。
:::
