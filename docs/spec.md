# 基本スペック

## マシン概要

| 項目 | 値 |
|------|-----|
| タイプ | AT機（セット継続型） |
| スピンコスト | 3 scrap × ベット倍率 |
| ベット倍率 | 1〜10倍 |
| 天井 | 800G → 天井CZ突入（設定共通） |
| AT初期ゲーム数 | 40G |
| AT終了演出 | 5G |
| 設定 | 1〜6（6段階） |

## 払い出し（3つ揃い）

| 図柄 | 配当（倍率） | 3 scrap 時の払い出し |
|------|------------|-------------------|
| Charcoal（炭） | 3倍 | 9 scrap |
| Wood（木材） | 4倍 | 12 scrap |
| Rope（ロープ） | 6倍 | 18 scrap |
| Bandage（包帯） | 7倍 | 21 scrap |
| Low Grade Fuel | 10倍 | 30 scrap |
| Sulfur Ore（硫黄） | 15倍 | 45 scrap |
| Scrap | 50倍 | 150 scrap |
| Lucky Seven（7） | 15倍 + AT突入 | 45 scrap + AT |

::: tip ポイント
**Lucky Seven（7揃い）** は配当15倍に加えて **AT「7 RUSH」** に突入する特別図柄です。通常時の7揃い確率は極めて低く（重み6/65536 ≒ 0.009%）、基本的にはCZ経由でATを目指します。
:::

## Low Grade Face 配当

Low Grade Fuel が **1つでも** 停止した場合、1倍（スピンコスト分）の配当が発生します。

## ゲームフロー

```
通常時（Normal）
  ├─ チェリー/スイカ → CZ抽選
  ├─ レア役 → 直撃AT抽選（4%）/ CZ抽選（60%）
  └─ 天井800G到達 → 天井CZ突入
       │
       ▼
  CZ「CHANCE ZONE」（5〜30G）
  ├─ 成功 → セット数獲得 → AT
  └─ 失敗 → 通常時へ
       │
       ▼
  AT「7 RUSH」（40G × セット数）
  ├─ レア役 → 上乗せ（AddOn）
  ├─ スイカ → スペシャルアドオン抽選
  ├─ AT判定 → 継続/終了判定
  └─ 終了 → AT Ending（5G）→ 通常時へ
```

## 内部モード

通常時は **4つの内部モード** が存在し、CZ突入率に影響します。

| モード | CZ突入期待度 |
|--------|------------|
| Low | ★☆☆☆ |
| Normal | ★★☆☆ |
| High | ★★★☆ |
| SuperHigh | ★★★★ |

モード移行は **AT終了時** と **CZ失敗時** に発生し、設定によって移行確率が異なります。

::: warning
内部モードは直接確認できません。挙動から推測する必要があります。
:::
