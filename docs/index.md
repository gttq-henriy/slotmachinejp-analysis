## サイト概要

本サイトは **Rust サーバー用スロットマシンプラグイン「SlotMachineJP」** の内部データを完全解析したものです。

パチスロ風のゲーム性を持つ本機の **設定差・天井・CZ/AT仕様** を網羅的にまとめています。

::: tip 解析データについて
本サイトのデータはプラグインの内部テーブル (v2.1.0) に基づいています。サーバー管理者がテーブルをカスタマイズしている場合、実際の挙動と異なる可能性があります。
:::
